Detailed Description

Subclass of BaseEntityState used for the fleeing state of an entity. In this state, an entity is moving away from a target entity.

PlayerEntity &  target 
)

Default constructor. Calls the super class constructor, passing in the name of this state. Initializes member variables for the source and target entity pointers.

Parameters
sourceA pointer to the source entity of this state. This is the entity that is currently in this state. We need to be able to access and modify the entity's position and velocity, according to the target's position.
targetA reference to the target entity to flee from. We need access to the target's position, so we can move away from it.
