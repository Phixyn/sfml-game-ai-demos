◆ m_localtime

errno_t GameAIDemos::Logger::m_localtime = localtime_s(&m_timeInfo, &m_timeT)
private

localtime() converts a given time since epoch into calendar time, expressed in local time, in the struct tm format. The result is stored in a struct tm.

localtime_s() is Microsoft's thread-safe version of this function. Its first argument is a pointer to the time structure to store the result in. The second argument is a pointer to the time since epoch time_t to be used.
